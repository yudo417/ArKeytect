## 💻 システム条件 / Overview

| プラットフォーム | 対応バージョン |
|------------------|----------------|
| **macOS** | macOS 13.0 (Ventura) 以降 |

> **注意**:
> - 本アプリは**メニューバーの常駐アプリ**です
> - **アクセシビリティ権限**が必要です（初回起動時に許可を求められます）
> - Nintendo Switch Pro Controllerが必要です


## 📋 特徴 / Feature

ArKeytectは，Switch - ProControllerを使ってMacを快適に操作できるようにする以下の機能を提供しています

1. **柔軟なキーマッピング**
   - Pro Controllerの各ボタンにキーボードのキーやマウス操作を割り当て
   - 修飾キー（Command, Option, Control, Shift）にも対応

2. **プロファイル・レイヤーシステム**
   - 用途別に複数のプロファイルを作成
   - レイヤー機能でZR/ZLボタン押下時の設定を切り替え
   - ZR+ZL同時押しで専用レイヤーへの切り替えも可能

3. **スティック・カーソル操作**
   - 左スティックでマウスカーソル移動
   - 右スティックでスクロール操作
   - 感度調整可能

## 🕹️ 操作 / Operation
### 🖱️ メニューバーでの操作について

macのメニューバーアイコンは、**左右クリックで挙動が異なります**．

- **左クリック**：  
  ArKeytectの機能の[オン/オフ]の切り替え

- **右クリック**：  
  メニューが開き，設定や終了など選択できます
